## gsap.fromTo()



Allows you to define BOTh the starting and ending values for an animation

```ts
//animate ".box" from an opacity of 0 to an opacity of 0.5
gsap.fromTo(".box", { opacity: 0 }, { opacity: 0.5, duration: 1 });
```
